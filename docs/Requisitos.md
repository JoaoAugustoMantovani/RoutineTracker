# Requisitos Funcionais

## Módulo de Atividades
- RF01: O sistema deve permitir o cadastro de Tarefas.
- RF02: O sistema deve permitir o cadastro de Rotinas com padrões de repetição personalizados.
- RF03: O sistema deve permitir o cadastro de Compromissos com horários fixos.
- RF04: O sistema deve permitir a configuração de notificações prévias dinâmicas.

## Módulo de Performance
- RF05: O sistema deve permitir o registro de sessões de treino.
- RF06: O sistema deve gerar relatórios de progressão baseados nas sessões registradas.


# Requisitos Não Funcionais

- RN01: O sistema não deve excluir registros, adotando a inativação lógica (Soft Delete) através do campo `ativo`.
